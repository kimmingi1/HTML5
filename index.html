🌐 HTML 주요 태그 정리

웹 프로그래밍의 기초가 되는 HTML 태그들을 알아봅시다.

1. 텍스트 관련 태그

<h1> ~ <h6>

제목을 나타낼 때 사용합니다. 숫자가 작을수록 글자가 큽니다.

예시: <h1>대제목</h1>

<p>

일반적인 문단(Paragraph)을 나눌 때 사용합니다.

예시: <p>내용을 입력합니다.</p>

2. 멀티미디어 & 링크

<img>

이미지를 삽입합니다. src 속성으로 경로를 지정합니다.

예시: <img src="img/photo.jpg" alt="설명">

<a>

다른 페이지로 이동하는 링크를 만듭니다.

예시: <a href="https://google.com">구글로 이동</a>
